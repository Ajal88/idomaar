Idomaar - CrowdRec Reference Framework

The CrowdRec reference framework is an implementation of algorithms that have been developed within CrowdRec and serves to allow the testing of these algorithms before they are deployed in the real-world for large-scale user tests.

Download Idomaar

Reference Framework is developing within the FP7 European project CrowdRec.

Github Project

Reference Framework Features

The CrowdRec reference-framework is an open source framework that contains algorithms that are developed in the course of European FP7 project CrowdRec.

Get involved

Reference Framework is hosted on GitHub and open for everyone to contribute.
Please give us some feedback and join the development!

Contribute

You want to help us and participate in the development or the documentation? Just fork reference-framework on GitHub and send us a pull request.

Found a bug?

Open a new issue on GitHub. Please search for existing issues first and make sure to include all relevant information.

Used reference-framework?

You used the reference-framework while developing your project or writing your paper? We would love to know about it, send us an email or ping us on twitter.
